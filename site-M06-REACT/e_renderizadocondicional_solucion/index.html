    Ejercicio: Mostrar información condicional de un número#

    Enunciado#

    Crea un componente llamado AnalizadorNumero que reciba una prop valor de tipo number.

    Este componente debe analizar el número recibido y mostrar condicionalmente la siguiente información, solo si aplica:

    • Si el número es positivo o negativo
    • Si el número es par o impar
    • Si es múltiplo de 2, 3 o 4

    Cada condición debe mostrarse como un párrafo independiente solo si se cumple. Si no se cumple, ese párrafo no debe renderizarse.

    Objetivos técnicos#

    • Crear un componente AnalizadorNumero en components/
    • Definir las props en types/Numero.ts
    • Integrar el componente en la página HomePage
    • Pasar un número fijo (por ejemplo, 12) como prop para probar la lógica

    Código con huecos#

    types/Numero.ts#

    export type NumeroProps = {
  valor: number;
};

    components/AnalizadorNumero.tsx#

    import React from 'react';
import { NumeroProps } from '../types/Numero';

const AnalizadorNumero: React.FC<NumeroProps> = ({ valor }) => {
  return (
    <div>
      {valor > 0 && <p>El número es positivo</p>}
      {valor < 0 && <p>El número es negativo</p>}
      {valor % 2 === 0 && <p>Es par</p>}
      {valor % 2 !== 0 && <p>Es impar</p>}
      {valor % 2 === 0 && <p>Múltiplo de 2</p>}
      {valor % 3 === 0 && <p>Múltiplo de 3</p>}
      {valor % 4 === 0 && <p>Múltiplo de 4</p>}
    </div>
  );
};

export default AnalizadorNumero;

    pages/HomePage.tsx#

    import React from 'react';
import AnalizadorNumero from '../components/AnalizadorNumero';

const HomePage: React.FC = () => {
  return (
    <section>
      <h1>Inicio</h1>
      <AnalizadorNumero valor={12} />
    </section>
  );
};

export default HomePage;
