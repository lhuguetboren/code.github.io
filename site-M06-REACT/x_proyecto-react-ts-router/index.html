    Plantilla React 19 + TypeScript + React Router DOM#

    Este proyecto es una plantilla educativa pensada para enseñar a alumnos cómo estructurar una aplicación React moderna usando TypeScript y react-router-dom. Incluye:

    • Tipado completo con TypeScript
    • Estructura modular con componentes y páginas
    • Navegación con React Router DOM
    • Comentarios educativos para cada archivo

    Estructura del Proyecto#

    src/
├── components/
│   ├── Header.tsx
│   └── Footer.tsx
├── pages/
│   ├── HomePage.tsx
│   ├── AboutPage.tsx
│   └── NotFound.tsx
├── App.tsx
└── main.tsx

    Archivos y Código#

    📄 main.tsx#

    import React from 'react';
import ReactDOM from 'react-dom/client';
import { BrowserRouter } from 'react-router-dom';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <BrowserRouter>
      <App />
    </BrowserRouter>
  </React.StrictMode>
);

    📄 App.tsx#

    import { Routes, Route } from 'react-router-dom';
import Header from './components/Header';
import Footer from './components/Footer';
import HomePage from './pages/HomePage';
import AboutPage from './pages/AboutPage';
import NotFound from './pages/NotFound';

const App: React.FC = () => {
  return (
    <div className="app-container">
      <Header />
      <main className="main-content">
        <Routes>
          <Route path="/" element={<HomePage />} />
          <Route path="/about" element={<AboutPage />} />
          <Route path="*" element={<NotFound />} />
        </Routes>
      </main>
      <Footer />
    </div>
  );
};

export default App;

    ✨ Ejemplo: Uso de spread operator (...props) para pasar props dinámicas#

    Este patrón es útil para pasar automáticamente todas las props de un objeto sin escribirlas una por una.

    type CardProps = {
  titulo: string;
  descripcion: string;
};

// Componente que acepta props usando spread
const Tarjeta: React.FC<CardProps> = (props) => {
  return (
    <div className="card">
      <h3>{props.titulo}</h3>
      <p>{props.descripcion}</p>
    </div>
  );
};

// Uso desde el padre
const datos = {
  titulo: "Producto 1",
  descripcion: "Una descripción del producto"
};

<Tarjeta {...datos} />

    🏷️ Ejemplo: Props implícitas de HTML (atributos estándar)#

    Los componentes personalizados pueden usar atributos de HTML estándar como onClick, id, className, etc.

    type BotonProps = React.ButtonHTMLAttributes<HTMLButtonElement>;

const BotonPersonalizado: React.FC<BotonProps> = (props) => {
  return (
    <button className="mi-boton" {...props}>
      {props.children}
    </button>
  );
};

// Uso
<BotonPersonalizado onClick={() => alert("Click")} id="boton1">
  Haz clic aquí
</BotonPersonalizado>

    Este patrón permite que el componente se comporte como un botón normal, pero con estilos o lógica personalizada.
