    Ficha Técnica: Uso Básico de useState y useEffect en React#

    Descripción#

    En esta ficha se explican los conceptos básicos de los hooks useState y useEffect en React, incluyendo su funcionalidad, sintaxis y ejemplos prácticos.

    Archivos Necesarios y Estructura#

    • App.tsx: Componente principal donde implementaremos useState y useEffect.
    graph TD;
    A[Componente Principal] -->|Define Estado con useState| B[Estado del Componente]
    B -->|Cambio de Estado con setState| C[Renderizado del Componente]
    A -->|Ejecuta Efectos Secundarios con useEffect| D[Efectos del Componente]
    D -->|Se Ejecuta en Montaje o Cambio de Dependencias| E[Actualización de Estado]

    Conceptos Clave#

    1. ``: Manejo del Estado#

    El hook useState permite a los componentes funcionales manejar estados internos.

    Ejemplo Básico de ``#

    import React, { useState } from "react";

const Contador: React.FC = () => {
    const [contador, setContador] = useState(0);

    return (
        <div>
            <h1>Contador: {contador}</h1>
            <button onClick={() => setContador(contador + 1)}>Incrementar</button>
        </div>
    );
};

export default Contador;

    Explicación:

    • useState(0): Inicializa el estado contador con valor 0.
    • setContador(contador + 1): Actualiza el estado cuando se presiona el botón.
    • Cada actualización de estado provoca un re-renderizado del componente.

    2. ``: Manejo de Efectos Secundarios#

    El hook useEffect permite ejecutar efectos secundarios en componentes funcionales.

    Ejemplo Básico de useState#

    import React, { useState, useEffect } from "react";

const Temporizador: React.FC = () => {
    const [segundos, setSegundos] = useState(0);

    useEffect(() => {
        const intervalo = setInterval(() => {
            setSegundos((s) => s + 1);
        }, 1000);

        return () => clearInterval(intervalo);
    }, []);

    return <h1>Tiempo transcurrido: {segundos} segundos</h1>;
};

export default Temporizador;

    Explicación:

    • useEffect(() => { ... }, []): Se ejecuta una sola vez al montar el componente.
    • setInterval(): Inicia un contador que incrementa el estado cada segundo.
    • return () => clearInterval(intervalo): Limpia el intervalo cuando el componente se desmonta.

    3. Combinando useState y useEffects#

    Podemos combinar ambos hooks para actualizar el estado con datos externos, por ejemplo, una API.

    Ejemplo: Cargar Datos desde una API#

    import React, { useState, useEffect } from "react";

const Usuario: React.FC = () => {
    const [usuario, setUsuario] = useState<{ nombre: string } | null>(null);

    useEffect(() => {
        fetch("https://jsonplaceholder.typicode.com/users/1")
            .then((res) => res.json())
            .then((data) => setUsuario({ nombre: data.name }));
    }, []);

    return <h1>Usuario: {usuario ? usuario.nombre : "Cargando..."}</h1>;
};

export default Usuario;

    Explicación:

    • useEffect llama a la API al montar el componente.
    • setUsuario({ nombre: data.name }) almacena la respuesta en el estado.
    • El usuario se muestra cuando los datos se han cargado.

    Conclusión#

    • useState permite manejar valores dinámicos dentro de un componente.
    • useEffect ejecuta efectos secundarios como peticiones a APIs o temporizadores.
    • Combinando ambos, podemos manejar actualizaciones y sincronización de datos eficientemente.
