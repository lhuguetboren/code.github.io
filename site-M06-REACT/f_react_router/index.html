    Ficha Técnica: React Router en Vite#

    Descripción#

    En esta ficha se explican los conceptos clave sobre el uso de React Router en proyectos creados con Vite para gestionar la navegación entre páginas.

    Archivos Necesarios y Estructura#

    • App.tsx: Componente principal que define las rutas.
    • pages/Home.tsx: Página de inicio.
    • pages/About.tsx: Página "Acerca de".
    • pages/Contact.tsx: Página de contacto.

    1. Instalación de React Router#

    Para habilitar la navegación en un proyecto de React con Vite, primero instalamos React Router:

    npm install react-router-dom

    2. Configuración de Rutas en App.tsx#

    import React from "react";
import { BrowserRouter as Router, Routes, Route, Link } from "react-router-dom";
import Home from "./pages/Home";
import About from "./pages/About";
import Contact from "./pages/Contact";

const App: React.FC = () => {
    return (
        <Router>
            <nav>
                <Link to="/">Inicio</Link> |
                <Link to="/about">Acerca de</Link> |
                <Link to="/contact">Contacto</Link>
            </nav>

            <Routes>
                <Route path="/" element={<Home />} />
                <Route path="/about" element={<About />} />
                <Route path="/contact" element={<Contact />} />
            </Routes>
        </Router>
    );
};

export default App;

    Explicación:

    • BrowserRouter: Proporciona la funcionalidad de enrutamiento basada en URL.
    • Routes y Route: Definen qué componente se renderiza en cada URL.
    • Link: Se usa en lugar de <a> para evitar recargas de la página.

    3. Creación de Componentes de Página#

    Página de Inicio (Home.tsx)#

    import React from "react";

const Home: React.FC = () => {
    return <h1>Página de Inicio</h1>;
};

export default Home;

    Página "Acerca de" (About.tsx)#

    import React from "react";

const About: React.FC = () => {
    return <h1>Sobre Nosotros</h1>;
};

export default About;

    Página de Contacto (Contact.tsx)#

    import React from "react";

const Contact: React.FC = () => {
    return <h1>Contacto</h1>;
};

export default Contact;

    4. Ejecución y Pruebas#

    Para probar la aplicación en el navegador, ejecuta:

    npm run dev

    Abre la URL que aparece en la terminal (por defecto http://localhost:5173/) y navega entre las páginas usando los enlaces.

    Conclusión#

    • React Router permite gestionar la navegación entre páginas en aplicaciones SPA.
    • El uso de Link optimiza la navegación sin recargar la página.
    • Cada ruta carga un componente específico para mejorar la organización del código.
