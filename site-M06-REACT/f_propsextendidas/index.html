    Uso de Props exetendidas en React

    ✅ Ejemplo: Componente MiInput con props extendidas#

    import React from "react";

type MiInputProps = {
  etiqueta: string;
} & React.InputHTMLAttributes<HTMLInputElement>;

const MiInput: React.FC<MiInputProps> = ({ etiqueta, ...rest }) => {
  return (
    <div>
      <label>{etiqueta}</label>
      <input {...rest} />
    </div>
  );
};

export default MiInput;

    🔍 Uso del componente:#

    <MiInput
  etiqueta="Nombre:"
  placeholder="Escribe tu nombre"
  type="text"
  className="input-estilo"
  value="Juan"
  onChange={(e) => console.log(e.target.value)}
/>

    ✅ ¿Qué ganás?#

    • Puedes seguir usando todas las props estándar de <input> (type, placeholder, onChange, etc.).
    • Podés añadir props personalizadas como etiqueta.
    • El tipado de TypeScript sigue fuerte y seguro.
