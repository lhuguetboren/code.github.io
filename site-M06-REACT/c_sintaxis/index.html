    Sintaxis en React#

    Aquí tienes una lista de las principales declaraciones y estructuras de código utilizadas en React: ⸻

    1️⃣ Declaración de Variables

    React usa JavaScript moderno, así que se utilizan:

    • const (para valores inmutables)
    • let (para valores que cambian)
    • var (⚠️ evitar su uso)
    const nombre = "Juan";
let edad = 25;

    ⸻

    2️⃣ Importación y Exportación de Módulos

    React usa ES Modules (import/export) para manejar componentes y otros archivos.

    ✅ Importar un módulo:

    import React from "react"; import MiComponente from "./MiComponente";

    ✅ Exportación nombrada:

    export function sumar(a, b) { return a + b; }

    ✅ Exportación por defecto:

    export default function MiComponente() {
  return <h1>Hola!</h1>;
}

    ✅ Importación de exportación nombrada:

    import { sumar } from "./utils";

    ✅ Importación de exportación por defecto:

    import MiComponente from "./MiComponente";

    ⸻

    3️⃣ Declaración de Funciones

    ✅ Funciones tradicionales

    function saludar() {
  return "Hola!";
}

    ✅ Funciones flecha (usadas comúnmente en React)

    const saludar = () => "Hola!";

    ✅ Funciones flecha con parámetros

    const sumar = (a, b) => a + b;

    ⸻

    4️⃣ Componentes en React

    En React, los componentes pueden declararse de dos maneras:

    ✅ Componente Funcional (moderno)

    function MiComponente() {
  return <h1>¡Hola, React!</h1>;
}

    ✅ Componente de Clase (anticuado)

    import React, { Component } from "react";

class MiComponente extends Component {
  render() {
    return <h1>¡Hola, React!</h1>;
  }
}

    ⸻

    5️⃣ Uso de Props (Propiedades)

    ✅ Props en función normal

    function Mensaje(props) {
  return <h1>{props.texto}</h1>;
}

    ✅ Props con desestructuración

    const Mensaje = ({ texto }) => <h1>{texto}</h1>;
    ✅ Uso del componente con props

    <Mensaje texto="Hola, mundo!" />

    ⸻

    6️⃣ Estado con useState (Hook)

    ✅ Declaración de estado

    import { useState } from "react";

function Contador() {
  const [contador, setContador] = useState(0);

  return (
    <div>
      <p>Contador: {contador}</p>
      <button onClick={() => setContador(contador + 1)}>Incrementar</button>
    </div>
  );
}

    ⸻

    7️⃣ Manejo de Eventos

    ✅ Eventos en React (camelCase y arrow functions)

    function Boton() {
  const manejarClick = () => alert("¡Clickeado!");

  return <button onClick={manejarClick}>Clic aquí</button>;
}

    ✅ Eventos con parámetros

    function Mensaje({ texto }) {
  const mostrarMensaje = (mensaje) => alert(mensaje);

  return <button onClick={() => mostrarMensaje(texto)}>Mostrar</button>;
}

    ⸻

    8️⃣ Renderizado Condicional

    ✅ Operador ternario (? :)

    function Mensaje({ esVisible }) {
  return esVisible ? <p>Mensaje visible</p> : <p>Mensaje oculto</p>;
}

    ✅ Operador lógico && (renderizado corto)

    function Mensaje({ esVisible }) {
  return <>{esVisible && <p>Mensaje visible</p>}</>;
}

    ⸻

    9️⃣ Listas y .map()

    ✅ Renderizado de listas con map()

    function Lista() {
  const frutas = ["🍎 Manzana", "🍌 Plátano", "🍓 Fresa"];

  return (
    <ul>
      {frutas.map((fruta, index) => (
        <li key={index}>{fruta}</li>
      ))}
    </ul>
  );
}

    🔟 Efectos con useEffect (Hook)

    ✅ Ejecutar código cuando el componente se monta

    import { useEffect } from "react";

function Componente() {
  useEffect(() => {
    console.log("El componente se ha montado");
  }, []);

  return <p>Hola</p>;
}

    ✅ Ejecutar código cuando cambia una variable

    import { useState, useEffect } from "react";

function Contador() {
  const [contador, setContador] = useState(0);

  useEffect(() => {
    console.log("El contador cambió:", contador);
  }, [contador]);

  return (
    <button onClick={() => setContador(contador + 1)}>
      Contador: {contador}
    </button>
  );
}
    ⸻

    🔢 Manejo de Formularios en React

    ✅ Formulario controlado (usando useState)

    import { useState } from "react";

function Formulario() {
  const [nombre, setNombre] = useState("");

  const manejarCambio = (e) => {
    setNombre(e.target.value);
  };

  return (
    <form>
      <input type="text" value={nombre} onChange={manejarCambio} />
      <p>Nombre: {nombre}</p>
    </form>
  );
}
