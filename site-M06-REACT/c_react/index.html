    Conceptos Clave en React#

    1. React y TypeScript#

    React#

    React es una biblioteca de JavaScript utilizada para construir interfaces de usuario interactivas y eficientes. Se basa en un sistema de componentes reutilizables que permiten la creación de interfaces escalables.

    TypeScript#

    TypeScript es un superset de JavaScript que agrega tipado estático, lo que permite mayor seguridad en el código, autocompletado y mejores herramientas de depuración.

    Ejemplo de un componente con TypeScript: 

    import React from "react";

type Props = {
  mensaje: string;
};

const Saludo: React.FC<Props> = ({ mensaje }) => {
  return <h1>{mensaje}</h1>;
};

export default Saludo;

    2. Componentes en React#

    Los componentes son la base de React y permiten dividir la interfaz de usuario en piezas reutilizables. Se pueden clasificar en: - Componentes Funcionales: Se escriben como funciones de JavaScript y utilizan Hooks. - Componentes de Clase: Se escriben como clases y gestionan su propio estado mediante this.state (obsoleto en React moderno).

    Ejemplo de un componente funcional: 

    const Boton = () => {
  return <button>Haz clic</button>;
};

    3. JSX#

    JSX (JavaScript XML) es una sintaxis que permite escribir HTML dentro de JavaScript, facilitando la creación de interfaces dinámicas.

    graph TD;
    A[JSX] -->|Escribe HTML en JS| B[Renderizado]
    B -->|Transformado por Babel| C[JavaScript Puro]

    Ejemplo de JSX: 

    const Mensaje = () => {
  return <p>Hola, mundo</p>;
};

    4. Estado en React (useState)#

    El hook useState permite manejar datos dinámicos dentro de un componente.

    graph TD;
    A[Componente] -->|Define Estado| B[useState]
    B -->|Cambia Estado| C[setState]
    C -->|Re-renderiza| D[UI Actualizada]

    Ejemplo de useState en acción: 

    const Contador = () => {
  const [contador, setContador] = useState(0);

  return (
    <div>
      <p>Contador: {contador}</p>
      <button onClick={() => setContador(contador + 1)}>Incrementar</button>
    </div>
  );
};

    5. Efectos Secundarios (useEffect)#

    useEffect permite ejecutar código cuando el componente se monta o cuando cambian ciertas dependencias.

    graph TD;
    A[Componente Montado] -->|Ejecuta Efecto| B[useEffect]
    B -->|Cambia Estado| C[setState]
    C -->|Re-renderiza| D[UI Actualizada]

    Ejemplo de useEffect para llamadas a API: 

    useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/todos/1")
    .then(response => response.json())
    .then(data => console.log(data));
}, []);

    6. Props en React#

    Las props (propiedades) permiten pasar información de un componente padre a un hijo.

    graph TD;
    A[Componente Padre] -->|Pasa Datos| B[Componente Hijo]
    B -->|Usa Props| C[Renderizado]

    Ejemplo de Props: 

    const Mensaje = ({ texto }: { texto: string }) => {
  return <p>{texto}</p>;
};

    React utiliza un Virtual DOM para mejorar el rendimiento al actualizar la UI.

    8. React Router#

    React Router permite la navegación entre páginas sin recargar la aplicación.

    Ejemplo de rutas en React Router: 

    import { BrowserRouter as Router, Route, Routes } from "react-router-dom";

const App = () => {
  return (
    <Router>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </Router>
  );
};

    9. Manejo de Eventos en React#

    React maneja eventos como onClick, onChange, onSubmit de manera similar a JavaScript, pero con sintaxis propia de JSX.

    Ejemplo de evento onClick: 

    const Boton = () => {
  const handleClick = () => {
    alert("Botón clickeado");
  };

  return <button onClick={handleClick}>Clic aquí</button>;
};

    10. Persistencia de Datos (localStorage)#

    React puede almacenar datos en localStorage para mantener información entre sesiones.

    useEffect(() => {
  localStorage.setItem("dato", "valor guardado");
}, []);
