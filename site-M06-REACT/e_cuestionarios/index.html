    Ejercicio: Generar un formulario dinámico desde un JSON#

    Enunciado#

    A partir de un archivo JSON que representa un cuestionario, crea un componente que genere automáticamente los campos de formulario necesarios.

    Usa un componente MiInput reutilizable para los campos de tipo text, y crea componentes equivalentes para select y check si es necesario.

    El formulario debe:

    • Mostrar el título general del cuestionario.
    • Renderizar los distintos tipos de entrada según el tipo (text, select, check).
    • Mostrar la pregunta como etiqueta.
    • Aplicar restricciones de longitud (min, max) si se indican.
    • Validar:
    • Edad mínima si es fecha de nacimiento.
    • Formato de email y dominio requerido si se especifica.

    💡 No es necesario implementar la validación completa, solo simular que el formulario puede capturar los datos correctamente.

  const { titulo, preguntas } = cuestionario as ______;

  return (
    <form>
      <h2>{titulo}</h2>
      {preguntas.map((pregunta: ______) => {
        if (pregunta.tipo === 'text') {
          return (
            <______
              key={pregunta.id}
              etiqueta={pregunta.pregunta}
              name={pregunta.id}
              minLength={pregunta.restricciones?.min}
              maxLength={pregunta.restricciones?.max}
            />

        }

          return (
            <div key={pregunta.id}>
              <label>{pregunta.pregunta}</label>
              <select name={pregunta.id}>
                {pregunta.opciones?.map((opcion) => (
                  <option key={opcion} value={opcion}>{opcion}</option>

          return (

              <legend>{pregunta.pregunta}</legend>
              {pregunta.opciones?.map((opcion) => (
                <label key={opcion}>
                  <input type="checkbox" name={pregunta.id} value={opcion} /> {opcion}
                </label>
              ))}
            </fieldset>
          );
