    Ficha Técnica: Publicar una Página Simple con React#

    Descripción#

    En esta ficha se describe el proceso básico para crear y desplegar una página web simple con React, mostrando un único texto. Se utilizará Vite para crear el proyecto, se configurará y se desplegará en GitHub Pages.

    Archivos Necesarios y Estructura#

    • index.html: Archivo base donde se inyectará la aplicación React.
    • App.tsx: Componente principal que renderiza el texto.
    • main.tsx: Archivo donde se monta la aplicación en el DOM.
    • package.json: Archivo de configuración del proyecto.

    Paso 1: Crear el Proyecto con Vite#

    Ejecuta el siguiente comando en la terminal para generar un nuevo proyecto React:

    npm create vite@latest mi-pagina --template react

    Luego, accede al directorio del proyecto: 

    cd mi-pagina
npm install

    Paso 2: Configurar React#

    Edita App.tsx para mostrar un texto simple:

    import React from "react";

const App: React.FC = () => {
    return <h1>¡Hola, esta es mi página en React!</h1>;
};

export default App;

    Paso 3: Montar la Aplicación#

    En main.tsx, renderizamos el componente en el DOM:

    import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

const rootElement = document.getElementById("root");
if (rootElement) {
    const root = ReactDOM.createRoot(rootElement);
    root.render(
        <React.StrictMode>
            <App />
        </React.StrictMode>
    );
}

    Paso 4: Ejecutar en Modo Desarrollo#

    Para verificar que todo funciona correctamente, ejecuta:

    npm run dev

    Abre la URL que aparece en la terminal (por defecto http://localhost:5173/).

    Paso 5: Compilar para Producción#

    Para generar la versión optimizada del proyecto:

    npm run build

    Esto creará una carpeta dist/ con los archivos listos para ser desplegados.

    Conclusión#

    • Creamos una aplicación React con Vite.
    • Mostramos un simple texto en pantalla.
    • Compilamos y publicamos en GitHub Pages.
    • Ahora tenemos una página en línea con React. 🚀
